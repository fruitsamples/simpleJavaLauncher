Read Me Notes to "simpleJavaLauncher" Demo and Sample Code

=============================================================================
This sample code shows how to properly instantiating the 1.4.x, and future JVM's from a CoreFoundation tool. Custom launchers are not supported when using JDK 1.3.1.
=============================================================================
The minimum runtime requirements for this Sample Code are:

- Mac OS X 10.2.3 or later
- JDK 1.4.x or later
